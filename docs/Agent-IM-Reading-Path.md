# Agent IM 阅读顺序（Onboarding）

> 目标：帮助新成员在 30-60 分钟内建立统一上下文。  
> 原则：先看“为什么”，再看“现在在哪”，最后看“怎么执行”。

---

## A. 先看方向（10-15 分钟）

1. [GaiaLynk Vision](./Agent-IM-Vision.md)  
   - 理解项目定位、核心价值与长期愿景。

2. [Agent IM 对外路线愿景（Public）](./Agent-IM-Public-Roadmap-Vision.md)  
   - 理解对外叙事与阶段节奏。

3. [Agent IM 长期路线（当前节点）](./Agent-IM-Long-Term-Roadmap-from-Current-Node.md)  
   - 理解当前阶段重点与后续演进路径。

---

## B. 再看现状（10-15 分钟）

4. [Agent IM 内部进展记录（2026-03-18）](./Agent-IM-Internal-Progress-Record-2026-03-18.md)  
   - 快速知道“已完成什么、为何这么做、证据在哪”。

5. [CTO 工作进展与下一步计划（总控）](./Agent-IM-CTO-Work-Progress-and-Next-Plan.md)  
   - 查看执行标准、门禁口径、Go/No-Go 判定逻辑。

---

## C. 最后看执行（15-30 分钟）

6. [Staging 部署：创始人配合指引](./Agent-IM-Founder-Staging-Guide.md)  
   - 从 0 到 1 的部署步骤与协作方式。

7. [官网生产部署清单（从 0 开始）](./Agent-IM-Website-Production-Deployment-Checklist.md)  
   - 官网上线操作清单与自检项。

8. [Release Gate 手动 Workflow Secrets 说明](./Release-Gate-Manual-Workflow-Secrets.md)  
   - 三连门禁触发与证据链规范。

9. [Contract Baseline 说明](./contracts/README.md)  
   - 合同基线初始化与 drift 维护流程。

## E. 建议阅读输出

读完后建议产出一页简报（内部）：

- 我理解的当前阶段目标（3 句话）
- 当前门禁放行标准（3 条）
- 我接下来两周要负责的工作项（3 条）
